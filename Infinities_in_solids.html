Infinities in Solids

 

When NDDO methods were applied to solids, an unusual error was found; this is described in "Application of the PM6 method to modeling the solid state" J. J. P. Stewart,  Journal of Molecular Modeling, Volume 14, Issue 6 , pp 499-535;  10.1007/s00894-008-0299-7 http://www.ncbi.nlm.nih.gov/pmc/articles/PMC2386531/

This error is insignificant when applied to molecules, polymers and later systems, but gives rise to an infinite error when applied to solids.  In practice, the infinite error is merely very large, because true infinite sums are not used in the program, nevertheless these large errors made it impossible to accurately predict solid state properties, and therefore the NDDO approximation was modified to eliminate the underlying error in the infinite summation.   

That an insignificant error in molecules, polymers, and layer systems should give rise to an infinite error in a solid is counterintuitive, so a brief explanation of this phenomenon is warranted.  In the journal article, the origin of the infinite error was traced to a series expansion that can best be described as the sum of the reciprocals of the integers.  This specific series expansion is found only in true three-dimensional solids that use the standard NDDO two-electron two-center approximation.

Consider the function: 0.001(1/2 + 1/3 + 1/4 + 1/5, etc. )

For any finite set (i.e., any molecule), the sum of of this series is obviously small. But for the infinite set, the sum is infinite, and this was the case in PM6 before the correction was made.

Proof:

Starting with the series: S = 1/2 + 1/3 + 1/4 + 1/5 + 1/6 + 1/7 + 1/8 + 1/9 + 1/10 + 1/11 + 1/12 + 1/13 + etc.  = Σi1/j

Grouping the terms three at a time, we get: S = (12 + 8 + 6)/24 + (42 + 35 + 30)/210 + (90 + 80 +72)/720 + (156 + 143 + 132)/1716 + etc.

Therefore: S = 26/24 + 107/210 + 242/720 + 431/1716 + etc

Therefore: S = (1 + 2/24) + (1/2 + 2/210) + (1/3 + 2/720) + (1/4 +2/1716) + etc.

Therefore: S = (1 + bits) + 1/2 + 1/3 + 1/4 + etc.

Therefore: S = S + (1 + Σj (2(3j-2)!/(3j+1)!))

But any number that is equal to a non-zero number plus itself cannot be finite.

Therefore S = infinity

For a finite set, if Σi1/j is summed from j = 2 to j = 1000, the value of the sum is 6.48, and the corresponding integral, ∫x=2x=1000 1/x dx is Ln(1000) -Ln(2) =  6.21; the difference, 0.27, being more-or-less a constant after about 70 entries.

Let's explore this sum.  In order to have the finite set amount to a given integer, j, 2ej terms would be necessary.  For the first 20 integers, the number of terms is given in the adjacent table.

The significance of these numbers is clear:  Even if a molecule contains thousands of atoms, the sum of the set would only be in single digits.  For a true solid, however, where the number of terms is infinite, the value of the sum is also infinite.  In practice, finite sums are used, so the errors are finite.  This does not mean that they are negligible - again, in practice, they amount to several kcal/mol, essentially ruining the calculation.
