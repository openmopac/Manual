DIPOLE

By default animations, IRC, DRC, etc., show heats of formation for each geometry.  If DIPOLE is used, the ΔHf is replaced by the dipole.

A worked example showing the difference between the dipole and ΔHf animation is provided by the flexion of 1,4-dibromo-1,4-dicyanocyclohexane as it goes from the diaxial dibromo conformation to the diequatorial dibromo conformation via the boat conformation.  In Table 1 the following models can be identified in both the dipole and ΔHf animations.

Table 1: Stationary points on reaction path  
Model Dipole ΔHf   Description  
first 0.145 46.358   Starting axial conformation  
118 3.376 55.008 Transition state Half-boat, Br tilted axial  
216 3.863 50.782   Boat at end of axial path   
237 3.596 50.779   Boat at start of equatorial path  
293 3.291 55.682 Transition state Half-boat, CN tilted axial  
last 0.155 46.353   Ending equatorial conformation  
Table 2: Optimized stationary points  
ν cm-1 Dipole ΔHf   Description

i75.7 3.262 55.680 Transition state Half chair (Cs)

At the start of the animation (Model "first") the system should be in the optimized axial conformation, and have the point-group symmetry C2h
