(Modeling proteins)

1E66

(a) Catalytic triad hydrogenation incorrectly predicted using ADD-H

1E66 is an acetylcholinesterase containing a catalytic triad involving Ser200, Glu327, and His440.  ADD-H incorrectly makes the Glu327 neutral and puts a hydrogen atom on the nitrogen atom of the imidazole ring of His440.  Inspection of the log file shows that there is a fault involving Glu327, and His440, and examination of the catalytic triad reveals this fault.  This error is readily corrected by editing the ARC file from the ADD-H run.

(b) Problem with placement of various oxygen and nitrogen atoms

The log file of 1E66 also shows an unusual structure involving atoms C4607 and N4610.  C4607 is attached to an unrecognized residue (UNK468), and the nitrogen atom is not attached to any residue; instead, it forms an ammonia molecule.  In the original PDB structure, UNK468 is Arg468, but one of the nitrogen atoms of the guanidine group is wrongly positioned.  If structure "B" is selected (using ALT_A=B), then two nitrogen atoms are wrongly positioned.  This is an error in the original PDB file that should be corrected "by hand" - using a GUI - before adding hydrogen atoms. 

Other faults found in the PDB file involve the following atoms:

Atom   in     Residue
N2478   Lys325
N653   Arg88
 N654   Arg88
O1862   Glu247
O1863   Glu247
N2478   Lys325

As with the other errors, these errors should also be corrected before any further work is done with 1E66.

Original PDB entry dated: 08-AUG-00.

(c) Two methionine residues have unusually short S-CH3 bond-lengths

Both Met83 (C606) and Met520 (C4033) have unusually short bond-lengths, about 1.1Å instead of the expected 1.8Å.
