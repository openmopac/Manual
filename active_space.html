SPECIFIED SPIN COMPONENT NOT SPANNED BY ACTIVE SPACE

In a Restricted Hartree Fock (RHF) Configuration Interaction (CI) calculation, a specific spin state has been requested, but the size of the active space is too small.  To correct this, increase (or set, if an active space has not been requested) the active space.

An active space is the set of M.O.s used in the CI.  It is set using one of C.I.=n, C.I.=(n,m), and OPEN(n,m).

Keywords, such as SEXTET or Ms=2,  that require an active space do not create an active space; a keyword that defines the active space must also be supplied.
