Editing Macromolecules

 

The problems involved in editing macromolecules are different from those involved in editing small molecules.  In small systems an edit normally involves only a few atoms being moved, added, or deleted.  Such operations can easily be done using a GUI that has editing capability.  When macromolecules need to be edited, the number of atoms that are modified can reach into the hundreds, this makes editing much more difficult.  In an attempt to make this task easier, three very different methods have been developed.  The choice of which one to use to edit a particular system depends on what needs to be done and on the system.  The following descriptions provide a guide to making this choice:

"Cut and Paste" following a GEO_REF 0SCF HTML run

Use of JSmol's "MINIMIZE" function to correct errors.

Use of CVB to correct errors in starting structure
