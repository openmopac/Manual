MEP=2

The Orozco-Luque molecular electrostatic potential map for the Connolly surface is generated and printed. As with MEP=1, the output is written to <filename>.mep and <filename>.tab.

In order to use MEP=2, extra data are needed at the end of the normal data-set. The extra data consist of one line of numbers, as defined in the Table.

Layout of data: A B C D  

Table:

 Extra Data Required by MEP=2

Datum

Value

Name

Description

A

1.4

SCALE

Scale factor which determines the size of the surface. The radii of the spheres (centered on each atom) are determined by applying this factor to the van der Waals' radii.

B

0.2

SCINCR

Separation (in Ångstroms) between surfaces

C

0.5

DEN

Density of points on each surface

D

1

NSURF

Number of surfaces
