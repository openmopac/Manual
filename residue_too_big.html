Error message when using RESEQ

Sometimes, particularly when there is an unusual structure in a protein, the resequencing process makes a mistake.  When this happens an error message will be printed.  How to correct this fault depends on the type of error encountered.

If keywords ADD-H or SITE are present, add NORESEQ.  This prevents the resequencing of the atoms, and therefore avoids the error.  Edit the resulting file and try running with RESEQ - this often works because the hydrogen atoms provide information that helps identifying residues.

If RESEQ does not work, use a Graphical User Interface to look at the protein in the region of the atom indicated in the output.  Use CVB to make or break bonds so that the unusual feature looks like a normal feature, e.g., like a residue plus a hetero group, for example.
